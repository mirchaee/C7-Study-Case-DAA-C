# C7 Study Case DAA-C

Travelling Salesman Problem (TSP) solved using Brute Force Algorithm.

## Features
- Generate all permutations of cities
- Calculate total travel distance
- Find optimal route
- Time Complexity Analysis
- Space Complexity Analysis

## Input Example
[
[0,10,15,20],
[10,0,35,25],
[15,35,0,30],
[20,25,30,0]
]

## Algorithm
Brute Force (Permutation)

## Complexity
- Time Complexity: O(n!)
- Space Complexity: O(n)
